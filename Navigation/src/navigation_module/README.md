# navigation_module

## Описание

Пакет реализует **алгоритмы навигации** для двумерного следования по маршруту на основе GPS (координаты) и IMU (курс).

## Реализуемые функции

1. Преобразование WGS84 в локальные координаты кадра `map` (опорная точка `origin` маршрута).
2. Загрузка маршрута из JSON (waypoints, радиусы достижения).
3. Расчёт `linear.x` и `angular.z` для разворота и прямолинейного движения с коррекцией курса.

## Структура пакета


| Модуль | Реализация |
|--------|------------|
| core/navigation.py | Расстояние, пеленг, нормализация углов |
| core/gps_utils.py | WGS84 → локальные метры (ENU) |
| core/orientation.py | Кватернион → курс (yaw) |
| core/sim_waypoint.py | Команды поворота и прямого участка |
| core/filters.py | Низкочастотная фильтрация GPS (используется в navigation_ros) |
| mission/mission_frame.py | Локальная система координат маршрута (origin, yaw) |
| mission/waypoint_loader.py | Загрузка JSON-маршрута |
| config/sim_settings.py | Коэффициенты регуляторов и пороги |

## Формат маршрута

```json
{
  "origin": { "latitude": 55.0, "longitude": 37.0, "yaw": 0.0 },
  "default_reach_radius_m": 0.9,
  "waypoints": [
    { "id": "wp_1", "latitude": 55.0, "longitude": 37.0002 },
    { "id": "wp_2", "x": 10.0, "y": 5.0 }
  ]
}
```

Waypoints — в широте/долготе или в локальных x/y. `origin.yaw` — ориентация кадра `map`; `reach_radius_m` — радиус достижения точки.

## Алгоритмы управления


| Функция | Реализация |
|---------|------------|
| turn_twist | Разворот на месте (angular.z) |
| drive_twist | Прямой ход, коррекция курса (drive_heading_hold) |
| reached_waypoint | Проверка достижения точки |
| needs_course_realign | Повторный разворот при превышении порога курса |
