# Программный комплекс планирования маршрута покрытия и автономной навигации агроробота

Выпускная квалификационная работа: разработка программного комплекса для построения маршрута покрытия (CPP) сельскохозяйственного участка и проверки автономного движения гусеничного мобильного агроробота по заданным опорным точкам в симуляционной среде Gazebo Harmonic.

Комплекс состоит из двух частей: планирование маршрута на карте и автономная навигация по экспортированным опорным точкам. Между частями передаётся согласованное описание маршрута в JSON.

## Структура проекта

```text
./
├── Map/            # планирование маршрута покрытия
└── Navigation/     # автономная навигация в симуляции
```

| Каталог | Описание | Документация |
|---------|----------|--------------|
| [`Map/`](Map/) | Контур поля, построение маршрута, экспорт waypoints | [Map/README.md](Map/README.md) |
| [`Navigation/`](Navigation/) | ROS 2, Gazebo Harmonic, движение по waypoints, RViz 2 | [Navigation/README.md](Navigation/README.md) |

## Сквозной сценарий

1. В **Map** — задать участок, построить маршрут покрытия, экспортировать waypoints.
2. В **Navigation** — загрузить маршрут, запустить симуляцию, проверить автономное движение.

Сборка и использование каждой части — в соответствующем README внутри каталога.
